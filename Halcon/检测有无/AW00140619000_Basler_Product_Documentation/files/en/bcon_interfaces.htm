BCON Interfaces

BCON stands for Basler Connectivity, which represents the addition of Machine Vision features to established data transmission standards from the embedded industry (such as LVDS or MIPI CSI-2).

BCON ensures a reliable broadband transfer of image data and, in connection with Basler dart board level cameras, enables a lean image processing architecture for embedded applications.

Related Topics
