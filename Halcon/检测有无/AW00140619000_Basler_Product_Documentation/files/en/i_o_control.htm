I/O Control

Most Basler cameras and camera modules offer input and output (I/O) lines. The topics in this section provide information about the I/O lines.

The input lines allow you to control the camera, e.g., trigger image captures at particular points in time. The output lines allow you to retrieve information about the camera, e.g., when you should apply flash lighting.
