daA2500-14lc-MZ7010

The dart BCON for LVDS Development Kit (daA2500-14lc-MZ7010) helps you to integrate a Basler dart BCON for LVDS camera into embedded vision systems.

It contains a dart camera module, a processing board based on the Xilinx Zynq®-7000 AP SoC, and additional accessories. The development kit offers a complete sample setup for an LVDS-based camera connection.

Delivered Parts

The development kit contains the following components:

  • Basler dart daA2500-14lc camera module, S-mount
  • Evetar lens M12B0816W ƒ/1.6, f8 mm, 1/2"
  • Processing board (Avnet MicroZed 7010 SoM)
  • Carrier board (Basler MicroZed BCON for LVDS carrier card)
  • microSD card (preinstalled with, e.g., Linux operating system, pylon Camera Software Suite for Linux, FPGA image pipeline)
  • Flexible flat cable, 0.2 m
  • USB 2.0 cable (type A plug to Micro B plug), 1.8 m
  • GigE cable Cat 5e, 2.0 m
  • Power supply, 15 W, 5 V @ 3 A
  • International power cable plugs
  • Mounting kit (including camera mounting plate, spacer bolts, screws, and nuts)
  • dart Getting Started Guide
  • dart BCON for LVDS Development Kit Quick Install Guide

Hardware Installation

  • For hardware installation, refer to the dart BCON for LVDS Quick Install Guide. The guide is also delivered with the dart BCON for LVDS Development Kit.
  • For information about the Avnet MicroZed board, see  the MicroZed Hardware User Guide.

Software Installation

  • The development kit is shipped with preinstalled software. For information about customizing the software and developing your own embedded vision solutions, refer to the dart BCON for LVDS Development Kit – Development Guide.
  • For online community support and further hardware-specific documentation, visit the dart BCON for LVDS Development Kit project on Imaginghub.
